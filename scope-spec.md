# 機能一覧（Scope SPEC）

## In Scope（今回実装する機能）

| 機能名                  | 説明                                    | 完了定義（テスト可能な条件）                              |
| ----------------------- | --------------------------------------- | --------------------------------------------------------- |
| 筋トレ履歴表示          | APIから取得した筋トレ履歴を一覧表示する | ページ表示時に履歴が表示される                            |
| API連携                 | Route HandlerとReactを接続する          | `/api/training-records` からデータ取得できる              |
| loading / error / empty | 状態ごとの表示を行う                    | 読み込み中・エラー・データ0件時に適切な表示が出る         |
| コンポーネント分割      | UI部品を整理する                        | `TrainingCard.tsx` と `TrainingList.tsx` に分割されている |
| UI改善                  | 筋トレアプリらしい見た目にする          | カードUIで見やすく表示される                              |

---

## Out of Scope（今回は実装しない機能）

| 機能名         | 理由                           |
| -------------- | ------------------------------ |
| ログイン認証   | Day4ではuserId固定で十分なため |
| 通知機能       | MVP範囲外のため                |
| AIおすすめ提案 | コア機能完成後に追加予定       |
| グラフ分析     | 時間内実装が難しいため         |
| SNS共有        | 本課題の目的外のため           |
