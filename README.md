# Home Training Support

## 概要

やる気度を選ぶだけで、その日に行う筋トレを決められる家トレ支援アプリ。

## 目的

家で筋トレをする人が、迷わず継続できるようにする。

## 主な機能

- やる気度選択
- おすすめ筋トレ表示
- 実施履歴（匿名表示）
- 自分の履歴ボタン（今後改善予定）

## 使用技術

- Next.js
- TypeScript
- React

## 起動方法

````bash
npm install
npm run dev

http://localhost:3000
````
