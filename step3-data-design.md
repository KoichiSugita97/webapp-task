# Step3：データ設計

## Training

| 項目名       | 型     | 説明     |
| ------------ | ------ | -------- |
| id           | number | ID       |
| trainingPart | string | 部位     |
| exerciseName | string | 種目名   |
| weight       | number | 重量     |
| reps         | number | 回数     |
| sets         | number | セット数 |

## CommunityRecord

| 項目名  | 型     | 説明     |
| ------- | ------ | -------- |
| id      | number | ID       |
| user    | string | 表示名   |
| message | string | コメント |
| detail  | string | 実施内容 |
